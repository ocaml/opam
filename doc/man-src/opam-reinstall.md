# NAME

opam-reinstall - Re-install packages

# SYNOPSIS

*opam reinstall* \<package\>...

# DESCRIPTION

This command does *opam remove* followed by *opam install*.

# PARAMETERS

\<package\>...
:   Package(s) to be installed.
