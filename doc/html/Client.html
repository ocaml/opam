val init : Types.repository -> Types.OCaml_V.t option -> int -> unit
Initialize the client a consistent state. init repo alias oversion cores means:
  • repo is the initial repository description,
  • oversion is the version of the compiler. None means we are using the system-wide installed compiler.
  • cores is the number of cores
